# Homebrew tap for Lori

[Lori](https://lori-app.sh) is a lightweight terminal intellisense overlay for
macOS: inline completions for 120+ tools, your $PATH, and shell aliases.
Local, no AI, no cloud.

## Install

```sh
brew install --cask matheuschein/lori/lori
```

Then open Lori, and from its menu bar icon go to Preferences > Setup to grant
Accessibility and add the shell integration (fish or zsh). Open a new terminal
tab and start typing.

The app keeps itself up to date (Sparkle). `brew upgrade --greedy` also works.
